# Portafolio de Danny Alberto Bernard Martinez

Portafolio en React con selector de idioma ES/EN y estructura limpia.

## Requisitos
- Node.js 18+ recomendado

## Comandos
- npm install
- npm run dev
- npm run build
- npm run preview
- npm run deploy

## Personalizar
- Traducciones: src/data/translations.js
- Estilos: src/styles/index.css
- Secciones: src/sections/

## Deploy en GitHub Pages
1) Ejecuta `npm install`.
2) Haz push a `main`.
3) En GitHub: Settings → Pages → Build and deployment → Source: GitHub Actions.
